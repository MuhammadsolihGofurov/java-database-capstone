Issue #1: User Stories for Patient (Bemor)
As a Patient, I want to register and login to the system so that I can access my medical dashboard.

As a Patient, I want to search for doctors by their name or specialty to find the right healthcare provider.

As a Patient, I want to book an appointment with a doctor for a specific date and time.

As a Patient, I want to view my past and upcoming appointments.

Issue #2: User Stories for Doctor (Shifokor)
As a Doctor, I want to login to my portal to see my daily schedule.

As a Doctor, I want to view the list of patients who have booked appointments with me.

As a Doctor, I want to set or update my available time slots so that patients can book accordingly.

As a Doctor, I want to write and save prescriptions for patients after a consultation.

Issue #3: User Stories for Admin (Administrator)
As an Admin, I want to manage (Add/Update/Delete) doctor profiles in the system.

As an Admin, I want to oversee all appointments to ensure the clinic is running smoothly.

As an Admin, I want to access system reports to see daily or monthly clinic activity.
